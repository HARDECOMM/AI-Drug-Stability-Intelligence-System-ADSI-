# AI Drug Stability Intelligence System (ADSI)

An AI + IoT system that predicts pharmaceutical drug degradation
during transportation and storage using environmental monitoring
and machine learning.

## Problem

Many medicines degrade when exposed to heat and humidity
in supply chains, especially in tropical regions.

## Solution

ADSI combines:

- IoT sensors
- AI degradation prediction
- Cloud monitoring
- Risk scoring dashboard

## System Architecture

Sensor → API → Database → AI Model → Dashboard

## MVP Goals

- Monitor temperature
- Predict degradation risk
- Display stability risk score
