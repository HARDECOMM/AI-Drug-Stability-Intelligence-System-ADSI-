# ADSI Visualization Notes

## Purpose

These visualizations support the interpretation of the ADSI machine learning model and dataset.

## Generated Charts

### 1. Risk Distribution

Shows the number of drug exposure scenarios in each degradation risk category.

### 2. Temperature Exposure vs Risk

Illustrates how higher temperatures and longer exposure durations increase degradation risk.

### 3. Feature Importance

Displays the relative importance of each environmental and drug-related feature in the Random Forest model.

### 4. Humidity vs Exposure Hours

Shows environmental exposure relationships across the synthetic dataset.

## Use Cases

These charts can be used in:

* scientific papers
* competition presentations
* GitHub README documentation
* prototype demonstrations
