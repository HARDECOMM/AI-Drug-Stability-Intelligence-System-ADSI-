# ADSI Report Workflow

## Export CSV
Use the "Export CSV" button to download recent monitoring records for spreadsheet review, regulatory analysis, or archival storage.

## Print / Save PDF
Use the "Print / Save PDF" button to:
- print a hard copy report
- save the current monitoring view as a PDF

## Recommended Use
- Pharmacy supervisor review
- Warehouse monitoring report
- Regulator inspection support
- Batch incident documentation
